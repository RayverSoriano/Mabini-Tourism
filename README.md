Mabini Tourism Website

Overview
Mabini Tourism is a responsive website showcasing the tourism attractions, destinations, and cultural heritage of Mabini, Batangas. The website serves as a digital guide for tourists, featuring dive spots, beaches, resorts, and local attractions in this popular coastal municipality.

Live Website
URL: https://mabini-tourism-llic.vercel.app

Features

1. Comprehensive Tourism Information
- Dive Spots & Beaches: Detailed information about popular diving locations and beach destinations
- Resorts & Accommodations: List of resorts with descriptions and amenities
- Tourist Attraction*: Cultural and natural attractions in Mabini

2. User-Friendly Interface
- Responsive design that works on desktop, tablet, and mobile devices
- Clean, intuitive navigation with organized categories
- Attractive visual presentation with relevant imagery
- Fast loading times and smooth user experience

3. Visitor Resources
- Location information and directions
- Contact details for local tourism office
- Cultural and historical context
- Travel tips and recommendations
